1. Общие сведения

1.1 Цели проекта
Создать минималистичный, современный, двуязычный помодоро-таймер с режимом always-on-top, прозрачным оверлеем, GitHub-style статистикой и едиными настройками без вкладок.
Режим always-on-top (compact overlay) с опцией прозрачного фона
Двуязычный интерфейс RU/EN с мгновенным переключением
GitHub-style contributions grid для визуализации продуктивности
Кастомные темы (Томат, Океан, Лес, Фиалка)
Единая панель настроек без вкладок (scroll-лента с секциями)
Профили таймеров (Pomodoro, Deep Work, custom)
Отображение прогресса на таскбаре + системный трей
Менее 50 MB RAM, менее 80 MB инсталлятор

2.6 Настройки (Settings)
Все настройки в едином scroll-потоке без вкладок. Секции разделены тонкими линиями.
Секция Profile: пилл-переключатель профилей + кнопка «+ new».
Секция Timer: stepper-компоненты (+/-) для Work/Break/Long break/Rounds. Toggle: auto-start break, auto-start work, count backwards. Select: display mode.
Секция Appearance: точки выбора цвета (5 пресетов + custom). Слайдеры Opacity, Size. Toggle: показывать фон, секунды, кнопки, анимацию. Select: шрифт таймера. Live preview внизу.
Секция Sound: слайдер Volume. Toggle: звуковые уведомления, start/end sounds, repeat work sound. Компонент выбора файла с кнопками Play/Browse.
Секция Behavior: toggle always-on-top, minimize to tray, launch on startup, taskbar progress, desktop notifications. Настройка hotkey через клик+запись комбинации.
Секция Data: экспорт JSON/CSV, импорт из YAPA, сброс данных (красная кнопка с подтверждением).
